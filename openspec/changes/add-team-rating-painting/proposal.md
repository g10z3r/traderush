## Why

Игрокам нужен глобально видимый, атмосферный способ смотреть рейтинг команд прямо в мире, без открытия книги или команд чата. Vision проекта описывает картину рейтинга как системный предмет, который можно повесить на стену; после реализации книги рейтинга уже есть доменный источник порядка команд и snapshot-паттерн, которые можно переиспользовать для world-space отображения.

## What Changes

- Добавить отдельный Trade Rush предмет картины рейтинга с идентификатором `trade-rush:team_rating_painting`.
- Позволить игроку повесить картину рейтинга на вертикальную стену как большой настенный объект размером 4×4 блока, сопоставимый с крупнейшими vanilla paintings.
- Отображать на картине read-only рейтинг команд: место, название команды и количество очков.
- Использовать тот же порядок рейтинга, что и книга рейтинга: результат `TeamService.listTeams()`, то есть очки по убыванию, затем имя команды.
- Показывать только первые команды, которые помещаются в область картины; команды ниже видимого лимита намеренно не показываются, потому что у картины нет перелистывания страниц.
- Автоматически обновлять отображение картины при server-side изменении состояния команд, влияющем на рейтинг: начисление очков, создание, переименование или удаление команды.
- Масштабировать обновления как общий top-rating snapshot для клиентов, а не как отдельную сетевую рассылку на каждую размещённую картину.
- Сохранить поведение книги рейтинга: открытая книга продолжает получать свои обновления, но картина получает отдельный глобальный путь синхронизации для world-space отображения.
- Не добавлять на картину action-кнопки, поля ввода, scroll, страницы или функции управления командами.
- Не реализовывать в рамках этой change бесконечный сундук на спавне для получения картины.
- Добавить минимальные assets/localization/README-описание для картины рейтинга.

## Capabilities

### New Capabilities

- `team-rating-painting`: системный предмет и настенный world-space display рейтинга команд, показывающий верхнюю часть рейтинга и автоматически обновляющийся при изменении командного состояния.

### Modified Capabilities

Нет.

## Impact

- Новая регистрация предмета в `TradeRushItems` и новый creative/admin workflow entry.
- Новый platform-level world object для размещённой картины рейтинга: вероятно custom hanging entity с регистрацией entity type и client renderer.
- Новый client-side renderer для world-space отображения фона картины и строк рейтинга.
- Новый compact top-rating snapshot/payload для синхронизации картины рейтинга с клиентами без открытия UI.
- Расширение runtime/team-change broadcasting: при изменениях команд нужно сохранять состояние, обновлять открытые UI книги/management и отправлять актуальный snapshot картины online clients.
- Переиспользование `TeamService.listTeams()` как единого источника порядка команд и очков.
- Возможный небольшой refactor snapshot builder-ов книги рейтинга, чтобы избежать дублирования логики строк рейтинга.
- Новые localization strings, item/painting assets и документация в `README.md`.
